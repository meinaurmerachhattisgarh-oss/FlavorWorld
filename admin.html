⚠️ Firestore Security Rules — Fix Permission Errors
If push fails with "Permission denied", paste these rules in Firebase Console
Open Firebase Console →
Firebase Console → Firestore Database → Rules tab → Replace all text with this → Publish
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /dishes/{dishId} { allow read: if true; allow write: if request.auth != null; } match /logs/{logId} { allow read, write: if request.auth != null; } match /admins/{userId} { allow read, write: if request.auth != null; } } }
